# 合約轉換器 — 技術文件

> 悠勢科技股份有限公司 ｜ IPO 換約專案

---

## 專案概述

業務將擬好的「對業主合約」上傳至網頁，系統自動產生對應的「對悠勢合約」並下載。

| 項目 | 說明 |
|------|------|
| 網址 | https://contract-generator-7mf1.onrender.com |
| GitHub | https://github.com/FredaMao/contract-generator |
| 部署平台 | Render（Free 方案） |
| 語言 / 框架 | Python 3.11 / FastAPI |

---

## 轉換邏輯

```
對業主合約 (.docx)
  甲方 = 業主（空白）
  乙方 = 志昌 / 瀚昱 / 毅源
         ↓
  1. 偵測乙方公司名稱
  2. 更新合約標頭（出租人 / 承租人）
  3. 填寫甲方指定匯款銀行帳戶
  4. 替換簽名頁（立契約書人）
  5. XML 格式驗證
         ↓
對悠勢合約 (.docx)
  甲方 = 志昌 / 瀚昱 / 毅源
  乙方 = 悠勢科技股份有限公司
```

> 合約條款本文完全不動，只替換甲乙方資訊。

## 檔案結構

```
contract-generator/
├── main.py            # FastAPI 後端（路由、檔案上傳/下載）
├── converter.py       # 核心轉換邏輯（XML 操作）
├── requirements.txt   # Python 套件清單
├── render.yaml        # Render 部署設定
├── static/
│   └── logo.png       # 悠勢科技 Logo
└── templates/
    └── index.html     # 前端網頁介面
```

---

## 本機執行

```bash
# 安裝套件
pip install -r requirements.txt

# 啟動伺服器
uvicorn main:app --reload

# 開啟瀏覽器
http://localhost:8000
```

## 更新程式碼並部署

```bash

git push
```

Render 偵測到 push 後會**自動重新部署**（約 3～5 分鐘）。

## 注意事項

- **輸入格式**：僅支援 `.docx`，不支援 PDF
- **免費方案限制**：閒置 30 分鐘後進入休眠，第一次開啟需等待約 30～50 秒喚醒
- **資料安全**：合約檔案只在轉換過程中存於記憶體，不會儲存在伺服器
