## 功能總覽

系統分為兩個分頁：

| 分頁 | 功能 |
|------|------|
| **合約轉換** | 上傳對業主合約 → 自動產生對悠勢合約 |
| **產生新合約** | 填寫表單 → 從範本產生新合約 |

---

## 分頁一：合約轉換

### 轉換邏輯

```
對業主合約 (.docx)
  甲方 = 業主
  乙方 = 志昌 / 瀚昱 / 毅源
         ↓
  1. 偵測乙方公司名稱（志昌／瀚昱／毅源）
  2. 更新合約前文甲乙方稱謂
       甲方：業主 → 志昌/瀚昱/毅源
       乙方：志昌/瀚昱/毅源 → 悠勢科技
  3. 填寫「甲方指定匯款銀行帳戶」
  4. 替換簽名頁（立契約書人）
  5. XML 格式驗證
         ↓
對悠勢合約 (.docx)
  甲方 = 志昌 / 瀚昱 / 毅源
  乙方 = 悠勢科技股份有限公司
```

> 合約條款本文完全不動，只替換甲乙方資訊與銀行帳戶。

### 注意事項
- 僅支援 `.docx` 格式，不支援 PDF
- 系統自動識別合約中的公司名稱（志昌資產管理有限公司 / 瀚昱開發股份有限公司 / 毅源開發股份有限公司）
- 若合約中找不到上述公司名稱，會回傳錯誤提示
- 支援分潤版（段落帳戶格式）與租金版（表格帳戶格式）

## 分頁二：產生新合約

### 表單欄位說明

#### 合約設定
| 欄位 | 選項 | 說明 |
|------|------|------|
| 管理公司 | 志昌 / 瀚昱 / 毅源 / 悠勢 | 選悠勢時使用 uspace 專屬範本 |
| 合約類型 | 租賃型 / 分潤型 | 決定使用哪份範本 |

#### 停車場資料
| 欄位 | 說明 | 帶入表頭 |
|------|------|----------|
| 建物編號 | 例：A001 | ✓ |
| 建物名稱 * | 必填，例：信義停車場 | ✓ |
| 停車場地址 | 完整地址 | ✓ |
| 車位數 | 數字 | ✓ |
| 建物綁定電話 | 例：02-2345-6789 | ✓ |
| 所得代號 | 下方說明 | ✓ |
| 業務人員 | 負責業務姓名 | ✓ |

#### 所得代號（radio 按鈕）
| 類別 | 選項 |
|------|------|
| 個人 | □ 空地租賃(51L)　□ 建物租賃(51J) |
| 法人 | □ 00 發票 |
| 管委會 | □ 空地租賃(51L)　□ 建物租賃(51J)　□ 00 發票 |

> 勾選後同步至：合約內文對應段落（■ 打勾）＋ 表頭「所得代號：」欄位

#### 稅務（共用）
| 選項 | 表頭顯示 |
|------|----------|
| 租賃型 + 稅前 | 租賃稅前/手續費內含 |
| 租賃型 + 稅後 | 租賃稅後/手續費內含 |
| 分潤型 + 稅前 | 分潤稅前/手續費內含 |
| 分潤型 + 稅後 | 分潤稅後/手續費內含 |

### 合約檔名規則

```
[建物編號建物名稱]-甲方名稱-合約類型.docx

範例：
  租賃型 → [A001信義停車場]-王大明-停車位租賃契約書.docx
  分潤型 → [A001信義停車場]-王大明-停車位服務契約書.docx
```

### 範本檔案

| 範本 | 管理公司 | 合約類型 |
|------|----------|----------|
| template_rent.docx | 志昌／瀚昱／毅源 | 租賃型 |
| template_profit.docx | 志昌／瀚昱／毅源 | 分潤型 |
| template_rent_uspace.docx | 悠勢 | 租賃型 |
| template_profit_uspace.docx | 悠勢 | 分潤型 |

> 範本存放於 `自動產生合約範本/` 資料夾，使用 Jinja2（docxtpl）語法。

## 檔案結構

```
contract-generator/
├── main.py                  # FastAPI 後端（路由）
├── converter.py             # 合約轉換邏輯（XML 操作）
├── generator.py             # 合約產生邏輯（docxtpl 範本渲染）
├── requirements.txt
├── render.yaml              # Render 部署設定
├── static/
│   └── logo.png
├── templates/
│   └── index.html           # 前端介面（轉換 + 產生兩分頁）
└── 自動產生合約範本/
    ├── template_rent.docx
    ├── template_profit.docx
    ├── template_rent_uspace.docx
    └── template_profit_uspace.docx
```

## 注意事項

| 項目 | 說明 |
|------|------|
| 輸入格式 | 僅支援 `.docx`，不支援 PDF |
| 必填欄位 | 產生新合約時「建物名稱」為必填 |
| 所得代號 | 在產生器勾選後，合約內文與表頭均同步更新 |
| 日期格式 | 系統自動將西元年轉為民國年（顯示於合約內） |
| 乙方% 自動計算 | 分潤型填入甲方%後，乙方%自動補為 100 - 甲方% |
| 免費方案限制 | 閒置 30 分鐘後休眠，第一次開啟需等 30～50 秒喚醒 |
| 資料安全 | 合約檔案僅在記憶體中處理，不儲存於伺服器 |
